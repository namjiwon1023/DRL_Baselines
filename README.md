## Installation
----------------------------------------------------------------
### Anaconda install
```
conda install pytorch torchvision torchaudio cudatoolkit=11.0 -c pytorch
```

### pip install
```
pip install torch==1.7.1+cu110 torchvision==0.8.2+cu110 torchaudio===0.7.2 -f https://download.pytorch.org/whl/torch_stable.html
```
